Eventempo

December 2019 - April 2020

I built a UWP countdown timer app for Windows 10 devices. Some features include support for multiple event countdown timers, ease of use, event autosave, colorful customization, and notification alerts. I worked on this app during winter break of my freshman year in college, and released incremental updates to the app in the following months.

Tools Utilized: C# .NET (backend), XAML (frontend), Visual Studio
